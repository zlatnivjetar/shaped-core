# Client Logos

Place property-specific logo files here during deployment.

## Required files

| File              | Usage                                          |
|-------------------|------------------------------------------------|
| `direct-logo.png` | Provider badge for direct booking reviews      |

## Notes

- This folder lives at `wp-content/mu-plugins/shaped-client-assets/logos/` in production
- The plugin resolves logos via a fixed path — only the domain changes per client
- Recommended format: PNG with transparent background, minimum 48px height for retina
